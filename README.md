# AI Code Bot - بوت البرمجة الذكي 🤖

## نظرة عامة

**AI Code Bot** هو بوت ذكي متقدم مصمم ليكون **أقوى من أي مهندس برمجيات في العالم**! 🚀

### المميزات الرئيسية

- 🧠 **ذكاء اصطناعي متقدم** - فهم عميق للبرمجة والمشاكل المعقدة
- 💻 **إنشاء مشاريع كاملة** - من 1 إلى 100+ ملف مع قاعدة بيانات
- 🔧 **دعم لغات متعددة** - Python, JavaScript, PHP, Java, C++, SQL, HTML/CSS
- 🎯 **حلول ذكية** - تحليل المشاكل وتقديم حلول مبتكرة
- 📱 **واجهة جميلة** - تصميم احترافي زي ChatGPT
- 🗄️ **قواعد بيانات متقدمة** - تصميم محسن مع علاقات معقدة
- 🌐 **تطبيقات ويب** - مواقع احترافية مع واجهات جميلة
- 📱 **تطبيقات موبايل** - React Native و Flutter
- 🔌 **APIs احترافية** - RESTful APIs مع أمان عالي
- 🚀 **نشر تلقائي** - إرشادات النشر والاستضافة

## المتطلبات

### متطلبات النظام
- **PHP**: 7.4 أو أحدث
- **MySQL**: 5.7 أو أحدث
- **خادم ويب**: Apache أو Nginx
- **مساحة**: 100MB على الأقل
- **ذاكرة**: 128MB RAM على الأقل

### متطلبات PHP
```bash
# Extensions المطلوبة
- pdo

- json
- mbstring
- openssl
- curl
- fileinfo
```

## التثبيت

### 1. تحميل الملفات
```bash
# انسخ الملفات إلى مجلد الاستضافة
cp -r ai-code-bot/* /var/www/html/
```

### 2. إنشاء قاعدة البيانات
```sql
# سجل دخول MySQL
mysql -u root -p

# أنشئ قاعدة البيانات
CREATE DATABASE ai_code_bot CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

### 3. تعديل الإعدادات
```php
# عدل ملف config.php
define('DB_HOST', 'localhost');
define('DB_NAME', 'ai_code_bot');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
```

### 4. تشغيل التثبيت
```
# افتح المتصفح واذهب إلى
http://yourdomain.com/install.php
```

### 5. تسجيل الدخول
```
المستخدم: admin
كلمة المرور: admin123
البريد: admin@aicodebot.com
```
